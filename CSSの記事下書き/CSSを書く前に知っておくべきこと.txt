CSSでは主に　何をどう変えるのかを記述してあげます
まず、CSSを扱ってく上で欠かせない用語があるので紹介します

例えば文字を赤色にしたい要素があったとします。
Htmlで<div>で囲ってあげて
CSSで 
div,p{
 color:red;
 font-size: 80px;
}
とかいてあげると文字の色とフォントサイズを変えれます
ここでは1 color font-size をプロパティー、redや80pxを値と呼びます
つまり、何をがプロパティーでどう変えるのかが値っていうことですね
プロパティーと値で宣言と呼ぶ
複数の宣言があるときは後ろに;
つけいて宣言が終了したことを伝える
　一番最後はなくてもいいが、慣習的につける

[}は宣言ブロック
divたぐ　p　→セレクターという
複数セレクターを指定するときセレクターリストなどと呼ぶ

MDN　のサイトを紹介



CSSの宣言には
3種類ある
・内部スタイルシート<style>
div p{
color:red;
}
</style>
・外部スタイルシート
・インラインすいるしーと

インラインスタイル
→htmlのタグの中にかける
<div style="color":red; font-size: 80px;></div>
タグの属性を用いる
たまに使う

外部スタイル
style.cssなどを作る



外部スタイルシートを一番使う
可読性が高い
プログラミングにおいて、だれが見ても理解できて補完性が高いことが重要
